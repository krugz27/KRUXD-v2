This experiment explored how onboarding can change as users reveal intent signals.

Instead of one static walkthrough, the prototype adapted prompts based on initial actions and skipped irrelevant guidance.

The output was a rules framework for adaptive onboarding logic that can be reused across products with different complexity profiles.
