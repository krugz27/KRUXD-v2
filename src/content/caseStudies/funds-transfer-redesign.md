This project focused on reducing user mistakes in a high-risk transaction flow.

I redesigned form structure around user intent, improved confirmations, and clarified fee/timing language at key decision points. We tested multiple prototypes and iterated quickly with compliance stakeholders.

The shipped flow lowered transfer-related support requests while improving completion confidence.
